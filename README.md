## ✨ Features

### Real-Time Weather
- 🌡️ Current temperature and conditions
- 📍 Multi-city support with favorites
- 🔄 Pull-to-refresh functionality
- 📡 Offline mode with cached data

### Premium Features
- ☀️ Sun Card (sunrise, sunset, day length)
- 🟢 Air Quality Index (PM2.5, PM10)
- 💬 Smart daily weather summary (Polish)
- 🌧️ Hour-by-hour rain forecast (6 hours)

### Value-Driven Features
- 📈 **Zmiana Pogody** - Tomorrow vs Today comparison
- ✨ **Komfort Człowieka** - Smart recommendations
- 📊 **Weather Score** - 0-100 comfort index
- 🎯 **Widget Card** - Essential metrics at a glance

### Technical
- 📱 Motorola Edge 50 Fusion optimized
- 🔐 AsyncStorage persistence
- 🌐 Polish localization
- ⚡ 8-second build time
- 🚀 3-4 second startup

## 🚀 Quick Start

### Prerequisites
- Node.js 16+
- Android SDK / Gradle
- ADB (Android Debug Bridge)
- Motorola Edge 50 Fusion or compatible device

### Installation

```bash
git clone https://github.com/supermamkonto-lab/weather-app.git
cd weather-app

npm install
npm start
npx react-native run-android
```

## 🎯 Architecture

### Tech Stack
- **Framework:** React Native
- **Language:** TypeScript (Strict Mode)
- **State:** React Context API
- **APIs:** wttr.in (Weather), Open-Meteo (Air Quality)
- **Storage:** AsyncStorage
- **Build:** Gradle 9.3.1

### Project Structure
```
src/
├── context/       # State management
├── types/         # TypeScript definitions
├── utils/         # Business logic
└── components/    # UI components
tests/             # Unit tests
android/           # Android build system
```
